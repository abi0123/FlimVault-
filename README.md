# FlimVault 🎬

FlimVault is a movie browsing web application built using React.  
Users can explore trending movies, search for films, and view movie details through a clean responsive interface.


## Features

- 🔍 Search movies
- 📈 Trending movies section
- 📱 Responsive UI
- 🎥 Movie details page


## Tech Stack

- React
- Tailwind CSS
- TMDB API
- Vite

---

## Installation

Clone the repository:

```bash
git clone https://github.com/abi0123/FlimVault.git
```

Go to the project folder:

```bash
cd FlimVault
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
